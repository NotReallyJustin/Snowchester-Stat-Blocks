# Snowchester Stat Blocks
You've heard of "Infrastructure as Code" and "Detection as Code." Now get ready for "DND Stat Blocks as Code!"

Okay on a more serious note, Github is where I keep all my stat blocks. I also promised by party that after the campaign is over, I'll make these stat blocks public so they can take a look at it and have fun with it.
* If you're part of the `Snowchester` campaign, hi!
* If you're not, hopefully you can find some of these stat blocks useful in your own campaigns. This is my first time DMing and building a proper world w/ custom stat blocks.

## Assumptions
The campaign starts from lv.3 - lv. 7/8/9. Martials get a +1 weapon around lv. 5. Mid-Magic setting. 

The timeline for this campaign is also significantly more modern than most campaigns. Most of the world is in the Renaissance Era, and IronVale is post-Industrial Revolution. There's some Sci-Fi stuff happening since the campaign revolves around Time.
